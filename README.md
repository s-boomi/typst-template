# Typst template

This is a template for Typst documents. As a prerequisite, it's recommended to install [mise](https://mise.jdx.dev/) as a tool for it to work. Installing Typst CLI as a standalone tool also works. Follow [Typst's open-source documentation](https://typst.app/open-source/#download) for specific OS installation instructions.

To use it, simply type the following command:

```command
git clone --depth=1 https://github.com/s-boomi/typst-template.git ./my-document && rm -rf ./my-document.git && cd ./my-document && mise trust
```

Then edit the `main.typ` file in `src` to get started. 

**NOTE:** It's heavily suggested for VSCode users to install the recommended extensions.
